Workshop · HTML Semántico

En este workshop aprenderemos a estructurar páginas web usando exclusivamente etiquetas semánticas de HTML5, mejorando la accesibilidad, el SEO y la legibilidad del código.

¿Qué es el HTML Semántico?

El HTML semántico utiliza etiquetas que describen el significado del contenido, no solo su apariencia. Etiquetas como <header>, <main>, <article>, <aside> y <footer> comunican la estructura al navegador y a los lectores de pantalla.

📅 UTN · 2026 · Workshop #1

Ventajas del marcado semántico

Mejora el posicionamiento SEO, facilita la accesibilidad para personas con discapacidad visual y hace que el código sea más fácil de mantener y entender por otros desarrolladores.

📅 UTN · 2026 · Workshop #1

Formulario semántico con <label> e <input>

Un formulario accesible siempre asocia cada <input> con un <label> usando el atributo for. El tipo input[type="search"] indica semánticamente que el campo es de búsqueda.
